# WAMA Control Unit

Open Source washing machine control unit by Fairdevices.

## Overview

The WAMA Control Unit is an open, repairable and modular control system for washing machines and similar household appliances.

The goal of this project is to replace proprietary and non-repairable electronics with a sustainable, documented and community-driven alternative.

## Project Status

🚧 Work in Progress

This project is currently under active development.

## Repository Structure

`software/`     Firmware source code

`hardware/`   Schematics and PCB files

`doc/`       Documentation

`production/`      Files necessary for the hardware realization (drill files, etc...)
